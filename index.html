Controls Help

Buttons

  • Shape icons: select drawing shape (rectangle, circle, triangle, pentagon, hexagon, bar).
  • Bomb: click canvas to spawn a timed bomb that explodes.
  • Clear: click any object to remove it.
  • Unbreakable (lock): new drawn objects become non-breakable.
  • Color picker: sets color for new shapes.
  • Pause/Resume: pauses or resumes physics simulation.
  • ? Help (?): opens this help window.

Sliders

  • Speed: controls physics simulation speed.
  • Strength: sets break resistance for newly created shapes.
  • Bomb: controls explosion power (radius/force/effects).

Tips

  • Strength and Unbreakable apply only to objects drawn after changing them.
  • Bomb and Clear are tool modes and are mutually exclusive with drawing mode.
